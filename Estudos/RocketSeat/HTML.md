
Hypertext Markup Language: texto que contém links
Markup: marcação do texto, elemento HTML ou tag. 
Exemplo:
````HTML
<p>conteúdo</p>
<!--comentátio -->
````

Atributos: adicionam informações e/ou configurações à uma tag
Exemplo:
````HTML
<a hreif"https://rocketseat.com.br">ver site</a>
<img src="image.jpg" />
<br/> <!--pular linha/>
````
